React Server Componentsはサーバー上でレンダリングされ、クライアントにJSを送らないコンポーネントです。

## メリット

バンドルサイズの削減やデータ取得の簡略化が期待できます。

```tsx
export default async function Page() {
  const data = await fetchData();
  return <div>{data}</div>;
}
```

## まとめ

適切に使い分けることでパフォーマンスの良いアプリケーションが作れます。
