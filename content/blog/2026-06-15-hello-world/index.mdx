## コードブロックの例

シンタックスハイライトの動作確認用に、TypeScriptのコードを掲載します。

```ts
type Post = {
  title: string;
  description: string;
  date: string;
  tags: string[];
  published: boolean;
};

function greet(post: Post): string {
  return `Welcome to "${post.title}"!`;
}

console.log(greet({
  title: "Hello World",
  description: "最初の記事です",
  date: "2026-06-15",
  tags: ["blog", "nextjs"],
  published: true,
}));
```

もう一つ、シェルコマンドの例も載せておきます。

```bash
npm install
npm run dev
```
